Preface

This software design document will outline the components for our Less Aggravating Role Play game. This software will feature GUI components. The GUI will allow the user to move around in a world where they can interact with chest's and fight enemy's.

The design will feature java classes that when put together will create our a fully functional game. The GUI components will make the game much more user friendly and make the game feel more immersive.
