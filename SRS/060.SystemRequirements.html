GUI Scenes Descriptions
1. Game Difficulty

1.1 Game will increment game difficulty to one higher difficulty each time the Player interacts with the difficulty option.

1.2 Game will reset game difficulty to the lowest difficulty when the Player interacts with the difficulty option and the game difficulty is currently at the maximum setting.

2. Play game option in main menu will

2.1 generate a new dungeon with the number of rooms and set enemy difficulty based on selected difficulty.

2.2 place Knight in the start room of the dungeon.
