Glossary

  • RPG - Role Playing Game
  • Player - The user that is interacting with the software
  • Knight - The character in the game that is contoled by the player
  • Dungeon - The world the player will be interacting with that contains many rooms
  • Room - An area within the dungeon that will contain chests, doors, and or a battle event
  • Chest - An object within a room that will contain a consumable or weapon
  • Door - An area within the room that will allow the player to enter another room
  • Consumable - An item in the player's inventory that will give a health boost or some other benefit for the user during battle
  • Game Difficulty - A game setting that the user can change that will effect the number of rooms the player can enter making it more or less difficult
  • Play Game - An option in the opening screen that allows the user start the game
  • Inventory - An object in the game that contains the player's consumables and weapons
  • Enemy - The characters players will battle in the rooms
  • Boss - The final enemy the player will battle to complete the game
  • Run Away - An option during the battle stage where the player has a chance of avoiding attacking the enemy
