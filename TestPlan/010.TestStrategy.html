Test Strategy

Team LARP will be using JUnit to test the Model and Controler portion of the MVC system architecture and user testing for the View portion. The style of testing will be white box. White box testing says that you can look at the code and examine all parts to make your decisions on what to test.



Unit Testsing

The unit testing will involve testing the individual classes and methods to ensure they don't crash and return what they are supposed to.

System Testing

The system testing will be done after all the components are finished. This will involve testing the system as a whole to make sure it meets the inital requirements determined in the first stage.

User Testing

The user testing will allow various users to use the system as it was intended to ensure the GUI componets and buttons respont the way they are intended
